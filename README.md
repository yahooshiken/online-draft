# online-draft
 Online draft tools. (Work in progress)

## Develop


1. **Start your mongo db.**

   ```shell
   brew services start mongodb
   ```

2. **Start server side apps.**

   ```shell
   # In /server directory
   yarn run dev

   # Also watch mode
   yarn run dev:watch
   ```

3. **Start client side apps.**

   ```shell
   # In /client directory
   yarn start
   ```
